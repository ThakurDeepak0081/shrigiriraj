

# Vindravan Shri Giriraj Sewa Sadan - Spiritual Hospitality Website

## Overview
A fully responsive, production-ready frontend website for a spiritual guest house in Vrindavan, featuring a devotional design aesthetic, animated hero section, and seamless user experience.

---

## Design System

### Color Palette
- **Primary Background**: White (85% usage)
- **Accent Colors**: 
  - Warm Yellow (devotion)
  - Peaceful Green (nature)
  - Spiritual Red (energy)
  - Royal Navy Blue (trust & depth)
  - Black (text contrast)

### Visual Style
- Soft shadows and rounded corners
- Smooth hover transitions
- Spiritual, peaceful, and holy atmosphere
- Clean typography with proper hierarchy

---

## Pages & Components

### 1. Global Components
- **Sticky Navbar**: Logo, organization name, navigation links (Home, Services, Gallery, Contact Us), mobile hamburger menu
- **Footer**: Connect section, address, phone numbers, useful links, social icons (Facebook, Twitter, YouTube)
- **Legal Modals**: Privacy Policy & Terms & Conditions as scrollable popups

### 2. Home Page
- **Hero Section**: 
  - Full-screen spiritual background image (Vrindavan temples/deities)
  - Subtle parallax or slow zoom animation
  - Semi-transparent navy/black gradient overlay
  - Organization name, motto, tagline
  - "Book Your Stay" & "Contact Us" CTAs
  
- **About Us Section**: Organization story, mission, values, and inspiration
  
- **Why Choose Us**: 6 feature cards highlighting tranquility, comfort, location, staff, reputation
  
- **Enjoy Your Stay**: Inspirational content section
  
- **Testimonials**: Static guest reviews with spiritual theming
  
- **Nearby Spiritual Places**: Cards for ISKCON Temple, Maa Vaishno Devi Dham, Banke Bihari Temple, Prem Mandir with distances
  
- **Enquiry CTA Section**: Call-to-action for bookings

### 3. Services Page
- Clean service cards showcasing:
  - Accommodation services
  - Peaceful spiritual stay
  - Guest assistance
  - 24x7 availability

### 4. Gallery Page
- Categorized room gallery with filters
- Room types: Non-AC, Double Bed AC, Four Occupancy, Multi-Occupancy
- Each card shows: Room type, AC status, bed type, capacity, description

### 5. Contact Us Page
- Organization details (address, phones, working hours)
- Map placeholder
- Smart contact form:
  - Subject dropdown (Booking / General Inquiry)
  - Conditional fields based on selection
  - Frontend-only validation with success feedback

---

## Interactive Features

### Booking Integration
- "Book Now" buttons open WhatsApp with pre-filled booking message
- Email CTA opens default mail client with mailto link

### Animations
- Hero background: Gentle parallax or slow zoom effect
- Section fade-in animations on scroll
- Smooth hover effects on cards and buttons
- Mobile-optimized performance

---

## Technical Implementation
- Clean component architecture with reusable components
- TypeScript for type safety
- Tailwind CSS with custom spiritual color palette
- React Router for navigation
- Responsive design for all device sizes
- SEO-friendly semantic HTML structure
- Accessible UI following best practices
